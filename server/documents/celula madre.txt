Los seres vivos estamos compuestos por células y sus productos de secreción.

La célula es la unidad básica estructural, funcional y biológica de los seres vivos.

Las células se unen formado tejidos. 

Los tejidos animales son el tejido conectivo, el tejido muscular, el tejido nervioso, y el tejido epitelial.

Los tejidos se unen formando órganos. El cuerpo humano tiene 79 órganos.

Los órganos se unen formando sistemas. Los sistemas del cuerpo humano son el sistema nervioso, el sistema reproductivo, el sistema digestivo, el sistema circulatorio, el sistema respiratorio, el sistema renal, el sistema musculo-esquelético, el sistema linfático, el sistema inmune, el sistema endocrino, y el sistema integumentario.

Los sistemas se unen formando el organismo.

Una célula especializada es la que ya tiene definido cual va a ser su función. Por ejemplo una célula de músculo cardíaco está definido que su función va a ser bombear sangre.

Una célula madre es una célula no especializada, o sea que su función aún está por definirse.

Todas las células nacen a partir de otra célula. Una célula madre es una célula no especializada, que puede dar origen a una célula diferenciada, o autorenovarse dando origen a otra célula madre idéntica.
